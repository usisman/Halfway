# Halfway (MVP)

Halfway, grupların adil bir buluşma noktası bulmasına yardımcı olan basit bir web uygulaması.

## Kısa notlar

- Backend: `apps/api` (Express)
- Frontend: `apps/web/public` (düz HTML/JS/CSS)
- Veri: bellek içi (DB yok)

## Hızlı başlangıç

```bash
make up
```

Sunucular:
- API: http://localhost:3000
- Web: http://localhost:5173

Kapatmak için:
```bash
make down
```
